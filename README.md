# Euzebios-Terror

Jogo de terror online em navegador com 10 fases, ranking compartilhado, jogadores conectados e controles para celular.

## Rodar no computador

```bash
npm start
```

Depois abra:

```text
http://127.0.0.1:4173
```

## Jogar pelo celular na mesma rede

Abra no celular o endereco de rede local mostrado pelo servidor, por exemplo:

```text
http://192.168.100.60:4173
```

O celular e o computador precisam estar no mesmo Wi-Fi.

## Publicar no Render

1. Suba este projeto para um repositorio no GitHub.
2. Entre em https://render.com.
3. Clique em New +.
4. Escolha Web Service.
5. Conecte o repositorio do GitHub.
6. O Render deve detectar o `render.yaml`.
7. Confirme o deploy.

O comando de inicio e:

```bash
npm start
```
